# DODOMIO Website

Trang chủ DODOMIO được xây bằng Next.js 16, React 19 và Tailwind CSS v4.

## Cấu trúc tối giản

```text
src/
  app/
    globals.css
    layout.tsx
    page.tsx
public/
  images/
    dodo/
      dodomio-logo.png
      dodomio.ico
      home.png
```

## Chạy dự án

```bash
npm install
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000)

## Build kiểm tra

```bash
npm run lint
npm run typecheck
npm run build
```
